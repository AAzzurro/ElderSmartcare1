# 快速开始指南

## 🚀 功能启用

新的"用药信息关联"功能已被集成到"问大夫"页面中。无需任何额外配置，功能即刻可用。

---

## 📱 老人端：如何使用

### 第一步：录入药品说明书
1. 打开应用，选择"👴 我是老人"
2. 进入"💊 药品说明书翻译" tab
3. 拍照或上传药品说明书图片（或输入文字）
4. 点击"提交给AI进行检测"
5. 核实AI识别的药品信息
6. 点击"➕ 确认无误，加入我的用药卡片"

### 第二步：用药咨询
1. 进入"👨‍⚕️ 问大夫" tab
2. 您会看到提示信息："您的用药记录将自动被检索，医生会根据您的实际情况给出专业建议。"
3. 输入您的问题，例如：
   - "我胃疼能吃布洛芬吗？"
   - "这个药和我现在吃的药能一起吃吗？"
   - "我血压高，这个咳嗽药能吃吗？"
4. 按 Enter 或点击提交
5. **查看医生的回答开头**：
   - ✅ 如果显示"🔍 已检索 2 条用药记录"，说明系统正确识别了您的用药信息
   - ✅ 医生会基于您的实际用药情况给出专业建议

### 示例互动

```
【您输入】：我胃疼，能吃阿司匹林吗？

【系统回答】：
🔍 已检索 2 条用药记录

您好！我已经查看了您的用药记录，发现您正在服用阿莫西林。这很重要，因为：

⚠️ 重要提醒：
- 阿莫西林本身对胃肠道的刺激相对较小
- 但如果您胃已经疼痛，不建议立即加用阿司匹林
- 两种药物合用可能加重胃部不适

我的建议：
1. 先吃点温和的食物或温水
2. 如果疼痛持续，请先停用阿司匹林
3. 咨询您的主治医生，确认是否能用其他胃药
4. 如果疼痛加重，请就医检查
```

## 👩‍⚕️ 护工端：如何使用

### 第一步：录入患者用药信息
1. 打开应用，选择"👩‍⚕️ 我是护工"
2. 进入"💊 信息录入" tab
3. **重要**：输入患者的床号和姓名（如："301床" 和 "王大爷"）
4. 拍照或上传药品说明书
5. 核实信息后点击"➕ 确认无误，加入机构排班"

### 第二步：为患者咨询医生
1. 进入"👨‍⚕️ 问大夫（护工）" tab
2. 在"患者床号"和"患者姓名"字段中输入具体的患者信息
3. 输入您的咨询问题，例如：
   - "患者晚上可以吃这种止咳药吗？"
   - "患者肝不好，这个药安全吗？"
   - "301床患者已经在吃高血压药，能加这个新药吗？"
4. 按 Enter 或点击提交
5. **查看医生的回答开头**：
   - ✅ 如果显示"🔍 已检索 1 条用药记录"，说明系统正确检索了该患者的用药信息
   - ✅ 医生会针对该患者的具体情况给出建议

### 示例互动

```
【护工输入】：
- 患者床号：301床
- 患者姓名：王大爷
- 问题：患者现在有点咳嗽，能吃止咳药吗？

【系统回答】：
🔍 已检索 1 条用药记录

我已查看王大爷的用药记录。他目前正在服用高血压药物，针对咳嗽问题的建议如下：

✅ 可以服用止咳药，但需要注意：
1. 避免含有"伪麻黄碱"的止咳药（可能升高血压）
2. 选择温和的蜂蜜止咳糖浆或非处方止咳药
3. 如果咳嗽持续超过一周，请联系医生
4. 建议在医生同意下再用药

⚠️ 特别提醒：
- 某些中成药止咳药中含有伪麻黄碱，会升高血压
- 给高血压患者用药前一定要确认成分
```

## 💡 使用建议

### 最佳实践

1. **老人端**：
   - 在提问前确保已添加所有当前服用的药品
   - 问问题时要具体，如"我吃的X药能和Y药一起吃吗？"
   - 医生的建议只供参考，重要决定请咨询真实医生

2. **护工端**：
   - 输入患者信息时尽可能使用床号或姓名（至少输入一个）
   - 如果患者有多个床号变更，请确保输入最新信息
   - 建议护工保留咨询结果的笔记，便于医生查看

### 常见问题

**Q: 为什么显示"未检索到用药记录"？**
A: 可能原因：
- 您还没有添加任何药品
- 护工端输入的床号或姓名与数据库中的不匹配（请检查大小写和空格）
- 药品已被删除或清空

**Q: 医生的回答没有提及我的用药信息，怎么办？**
A: 可能原因：
- 您的问题与用药无关（医生会基于问题内容灵活回答）
- 尝试用更具体的方式重新提问，如"我吃的X药能和Y一起用吗？"

**Q: 能否修改已录入的用药信息？**
A: 目前系统仅支持添加新记录，如需修改，您可以：
- 清空现有记录并重新添加
- 联系系统管理员
